# Screenshots

Drop your app screenshots in this folder so they show up in the main [README](../../README.md).

The README expects these filenames (rename yours to match, or update the README):

| Filename | What to capture |
|----------|-----------------|
| `dashboard.png` | The main dashboard with stats/charts |
| `live-feed.png` | Live webcam / CCTV feed with a detection overlay |
| `upload.png` | The upload page analyzing a video |
| `history.png` | The detection history table |

**Tip:** on Windows press `Win + Shift + S` to snip a region, then save it here.
Keep images reasonably sized (< 1 MB each, ~1280px wide is plenty).
